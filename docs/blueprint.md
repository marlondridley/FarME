# **App Name**: FarmFinder

## Core Features:

- Map View: Display nearby farms, farmers markets, and pop-up vendors on a map, similar to Life360.
- Location Posting: Enable farmers to post their location for the day.
- Real-time Updates: Allow farmers to update their location, available products, and delivery options in real-time.
- Search and Filter: Allow users to search and filter farms by product, location radius and delivery options.
- Favorite Farms: Allow users to follow favorite farms for quick access and updates on new crops and special events.
- Delivery Options: Farmers provide standard vs premium hand-picked delivery.
- Smart Crop Suggestions: Use an AI tool to suggest the best crop types to farmers for their business based on weather patterns, time of year, trending customer preferences, and geographic area.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F), reflecting the natural aspect of farming and fresh produce.
- Background color: Light cream (#F5F5DC), for a soft and inviting feel.
- Accent color: Soft brown (#A0522D) for highlighting key information and CTAs, providing a sense of warmth.
- Body and headline font: 'PT Sans' (sans-serif) for friendly, organic readability.
- Icons: Use leaves, barns, baskets, tractor, delivery box.
- Map-first design: The main screen displays an interactive map with pins indicating nearby farms, farmers' markets, and vendors.
- Subtle animations: Use animations for loading states and transitions to make the app feel dynamic but unobtrusive.
